## Installation

```xml
<dependency>
   <groupId>cl.daplay</groupId>
   <artifactId>lazylist</artifactId>
   <version>4.3.0</version>
</dependency>
```

```groovy
compile group: 'cl.daplay', name: 'lazylist', version: '4.3.0'
```
